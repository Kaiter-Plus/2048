# 2048

一个 2048 小游戏

## 安装依赖

```bash
npm install
```

如果你使用其它的包管理器(比如: bun | yarn | pnpm)，你可以使用(以 bun 为例)

```bash
bun install
```

## 运行

```bash
npm run dev
```

如果你使用其它的包管理器(比如: bun | yarn | pnpm)，你可以使用(以 bun 为例)

```bash
bun run dev
```

## 打包

```bash
npm run build
```

如果你使用其它的包管理器(比如: bun | yarn | pnpm)，你可以使用(以 bun 为例)

```bash
bun run build
```
